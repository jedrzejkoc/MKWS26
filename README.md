# Influence of Hydrogen Enrichment of Methane on Laminar Flame Speed and NOx Emissions

Final project for the *Computational Methods in Combustion* course.

Run `py flame_simulation.py` to execute all six parametric flame cases (H₂ fraction 0–50 vol.%). The script requires Python with `cantera`, `numpy`, and `matplotlib` installed (`py -m pip install cantera matplotlib`). Results are printed to the console and all six publication-quality plots are saved to the `plots/` directory. Once the simulation has run, compile `report.tex` with pdflatex (run twice to resolve references) to produce the final PDF report with the figures embedded.
